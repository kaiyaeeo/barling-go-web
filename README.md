# BARLING-GO 🚀

**BARLING-GO** adalah ekosistem digital terintegrasi yang dirancang untuk memperdayakan UMKM di wilayah Barlingmascakep (Banjarnegara, Purbalingga, Banyumas, Cilacap, dan Kebumen). Platform ini menggabungkan teknologi *Generative AI* dan *Geospatial Intelligence* untuk membantu UMKM lokal bertransformasi ke ranah digital, memperluas jangkauan pasar, dan meningkatkan efisiensi operasional.

## 🌟 Fitur Utama
*   **Dashboard Mitra UMKM**: Manajemen produk, pesanan, dan analitik toko yang *real-time*.
*   **Digital Tourism Ecosystem**: Integrasi promosi produk UMKM dengan potensi pariwisata lokal berbasis geospasial.
*   **AI-Powered Insights**: Analitik data penjualan untuk membantu pengambilan keputusan bisnis.
*   **Responsive User Experience**: Antarmuka yang dioptimalkan untuk perangkat *mobile* agar mudah diakses oleh mitra UMKM di mana saja.

## 🛠 Tech Stack
*   **Framework**: Next.js (App Router)
*   **Language**: TypeScript
*   **Styling**: Tailwind CSS
*   **Database & Auth**: Supabase
*   **Icons**: Lucide React
*   **Deployment**: Vercel
